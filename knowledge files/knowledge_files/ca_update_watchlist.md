## The point
This is the minimum set of pages that change often and can break people’s lives if we’re outdated.

### Identity documents
- CDPH Vital Records (birth/death/marriage/divorce pages)
- County Registrar-Recorder pages (LA County + big counties)
- SSA “Replace card” flow + SS-5 PDF

### DMV / ID
- DMV ID Cards page
- DMV Reduced/No Fee ID program pages
- REAL ID checklist + document list PDFs

### Medi-Cal
- DHCS Health Care Options (contact numbers, enroll/change plan)
- DHCS Medi-Cal Managed Care Health Plan Directory (county changes)

### LA County behavioral health
- LACDMH ACCESS line page
- LACDMH “Get Help Now”
- LA County DPH SAPC + SASH helpline materials

### Shelter / interim housing (LA)
- LAHSA “Find a Shelter” portal
- LAHSA interim housing list PDFs
- LAHSA safe parking list PDFs

### Quality triggers (auto-alerts)
- Link returns 404/500
- Phone numbers change
- “Last updated” date changes on PDFs
- Multiple user reports: “this is wrong” / “this number is dead”
